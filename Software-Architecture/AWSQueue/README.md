# :beginner:AWS Queue

基于AWS SQS的消息中间件，用Java语言编写一个发送程序和接受程序，实现“点对点”进程间通信。

Using  Message middleware based on AWS SQS and develop a sending client and a receiving client by Java, simulating the *P2P* mode
